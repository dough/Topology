Operations

closure of a point

All points having the same or fewer properties.

saturation of a point

All points having the same or additional properties.

closure of a set

Perhaps you would suggest it sould be all sets having the same of fewer properties, but that is only the case for certain topologies (the posets).

Technically we define closed set and then the the closure of a set is the intersection of all closed sets that include it; that is, the smallest closed set that includes the given set.

saturation of a set

Specialization

One entity is more specialized than another if it has more properties.` That is the whole purpose of taking the closure.

the specialization class of a topology

There is an entire set of topologies inducing the same closures (and saturations) of points as a given topology, called its specialization class.

the generic topology

This topology is generated by taking the given generic sets (closures of points) as a base for closed sets. It is the smallest topology having the same generic sets.

the saturation topology

This topology is generated by taking the unions of the given saturations (saturations of points) as a base for open sets. It is the largest topology having the same generic sets.
