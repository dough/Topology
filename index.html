the specialization class of a topology

There is an entire set of topologies inducing the same closures (and saturations) of points as a given topology, called its specialization class.

the generic topology

This topology is generated by taking the given generic sets (closures of points) as a base for closed sets. It is the smallest topology having the same generic sets.

the saturation topology

This topology is generated by taking the unions of the given saturations (saturations of points) as a base for open sets. It is the largest topology having the same generic sets.
